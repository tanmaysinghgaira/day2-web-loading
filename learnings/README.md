# Fullstack Classes - Learning Notes

This repository contains comprehensive notes from fullstack development classes covering JavaScript, HTTP, REST APIs, and backend development with Express.js.

## Structure

Learning materials are organized by day:

- **day09**: JavaScript Functions fundamentals
- **day10**: HTTP Standards & REST APIs
- **day12**: API Design for Resume Builder project
- **day13**: Express.js & Project Structure

## Topics Covered

### Day 09 - JavaScript Functions
- Function syntax and declarations
- Parameters and return values
- Function expressions and arrow functions
- Scope and best practices

### Day 10 - HTTP & REST APIs
- HTTP protocol fundamentals
- Request/Response structure
- HTTP Methods (GET, POST, PUT, PATCH, DELETE)
- HTTP Status Codes
- REST principles and best practices
- JSON data format

### Day 12 - API Design
- Resource-based API design
- RESTful endpoint planning
- Resource definitions for Resume Builder
- HTTP method mapping

### Day 13 - Express & Backend
- Express.js framework overview
- Project structure setup
- Route organization
- API endpoints for Resume Builder
- Testing with Postman

## Learning Outcomes

By studying these materials, you'll understand:
- Core JavaScript concepts
- Web communication protocols
- RESTful API design principles
- Backend development with Node.js and Express
- Project organization best practices
